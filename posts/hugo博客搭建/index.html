Hugo博客搭建

March 5, 2020

hugo

下载hugo

  1. 在 github 选择自己合适的版本下载，然后安装
  2. 将hugo添加到系统环境变量中
  3. 在bash中运行 hugo version 提示版本则说明安装成功

配置hugo

  1. 运行 hugo new site name 创建一个博客创建器，name为github用户名小写.github.io.creator

  2. 创建完成后进入文件夹，然后git init把文件夹作为一个git仓库

  3. 下载主题git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

  4. 然后，将主题添加到站点配置中：echo 'theme = "ananke"' >> config.toml

  5. 然后运行 hugo new posts/my-first-post.md 添加第一篇博客，名字可以自己更改

  6. 自定义主题，通过修改config.toml中的设置可以修改博客名字，博客主题，博客字体，博客地址

  7. 运行 hugo server -d 运行hugo静态页面，点击可以查看

将博客上传至github

  1. 运行hugo之后会在目录中生成一个public文件夹，将这个文件夹加入.gitignor中，然后在publi中运行git init git add .,将public作为一个仓库

  2. 在github 创建一个仓库，仓库名为github用户名.github.io

  3. 把public 上传到 这个仓库

  4. 在仓库中github pages设置到master保存

  5. 设置完成，博客网站地址为 你的用户名.github.io

更改博客皮肤

  1. 找想要的皮肤

  2. 在creator里创建theme文件夹

  3. 在文件夹中clone 皮肤地址，删除皮肤.git

  4. 更改config文件中的theme设置

  5. 启动hugo server -D

  6. 重新上传至github
